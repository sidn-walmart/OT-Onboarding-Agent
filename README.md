# ORBIT — OT Research, Boarding, Intelligence & Training

A Wibey AI agent purpose-built for the Walmart IoT/ OT Security team. It acts as an always-available SME, research assistant, and onboarding guide — combining internal Walmart knowledge sources with authoritative external OT intelligence platforms.

---

## Capabilities

| Mode | Trigger | What It Does |
|------|---------|--------------|
| **Answer** | OT/ICS questions | Searches Confluence, SharePoint, MITRE ATT&CK ICS, CISA, Dragos |
| **Find Docs** | "find", "search for", "locate" | Returns ranked results across all internal and external sources |
| **Summarize** | URL or page ID provided | Fetches and summarizes Confluence pages, SharePoint docs, CISA advisories |
| **Research Report** | "research", "deep dive", "full report" | Multi-source report with gap analysis and recommendations |
| **SSP Evaluation** | SSP file or "evaluate this system" | Runs `iot-ssp-evaluate-html` with MITRE/CISA/Dragos context woven in |
| **Onboarding** | "new to team", "how do I access", "catch me up" | Access guides, required training, tool ownership, recent meeting catch-up |
| **WCNP / Infra** | Kubernetes, pipeline, deployment questions | Delegates to `wcnp-t12r-agent`, supplements with internal docs |
| **Teach / Deep Dive** | "explain", "walk me through", IEC 62443, NIST 800-82 | Delegates to `mike` skill backed by full standard text |

## Microsoft 365 Integration

The agent has full read/write access to the Microsoft 365 suite via the `msgraph` skill (MS Graph API). This enables calendar-aware context, email triage, Teams chat retrieval, and meeting note lookup alongside OT security workflows.

| Platform | Capabilities |
|----------|-------------|
| **Outlook Email** | Read inbox, search emails, send/reply, move, delete, flag — useful for finding security notifications, incident emails, vendor correspondence |
| **Calendar** | List today's/week's meetings, create/update/delete events, check availability across attendees |
| **Microsoft Teams** | Read 1:1 and group chats, list team channels and messages, send messages — used to pull meeting chat history and Facilitator bot summaries |
| **Microsoft Loop** | Read `.loop` meeting note files stored in OneDrive/SharePoint — structured notes auto-generated by Teams Facilitator after meetings |
| **SharePoint** | Search sites and document libraries — primary source for OT policies, SSPs, standards, and onboarding materials |
| **OneDrive** | List, search, upload, download, and share files — includes meeting recordings, transcript files, and Loop files |
| **Microsoft Planner** | Read and manage team task boards — track sprint tasks, cross-team dependencies, and project milestones |
| **Microsoft To-Do** | Read and manage personal task lists — personal action items and flagged emails |
| **Azure AD / Directory** | Look up users by name or email — find team members, owners, and contacts across Walmart |

### Meeting Intelligence
The agent can reconstruct meeting summaries even when Microsoft Loop fails to generate a notes file, by reading the **Teams Facilitator bot's in-chat messages** (midpoint check-in, 6-minute warning, and end-of-meeting wrap). These are always present in the meeting chat thread regardless of Loop file creation status.

## Source Priority

Internal Walmart documentation is always checked before external sources.

---

## Output Format

All responses include:
- **Summary** — 2–4 sentence answer
- **Details** — full explanation with context
- **Sources** — every claim linked to a source
- **Recommended Actions** — actionable next steps
- **Confidence** — High / Medium / Low with rationale

CISA KEV entries are flagged `⚠️ KEV`. Stale internal docs (>6 months) are flagged `⚠️ Potentially stale`.

## Installation

Place `ot-onboarding-agent.md` in `~/.wibey/agents/` and restart Wibey. The agent will be available as `ot-onboarding-agent` across all Wibey sessions.
